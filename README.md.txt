# User stories

### User stories describe what a normal user should be able to do withing our application:

1. As a user, I want to add food and remove in the shopping cart from multiple restaurants.
2. As a user, I want to be able to cancel an order if the restaurant didn't start preparing it.
3. As a user, I want to see restaurants within my proximity.
4. As a user, I want to see a history of my orders.
5. As a user, I want to be able to use discount codes.
6. As a delivery driver, I want to receive the order details.
7. As a delivery driver, I want to be able to decline the order.
8. As a restaurant, I want to be able to add new products  in the menu.
9. As a restaurant, I want to be able to change the current price of products in the menu.
10. As a restaurant, I want to be able to remove current products from the menu.
